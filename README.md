Password Checker
A simple Python-based cybersecurity tool that evaluates password strength and provides improvement suggestions.
Features
- Checks password length
- Detects uppercase and lowercase letters
- Checks for numbers and special characters
- Gives password strength rating (Weak, Moderate, Strong)
Purpose
This project was built to understand basic cybersecurity principles related to password security and authentication safety.
Tools Used
- Python
- Regular Expressions (Regex)
Learning Outcome
Improved understanding of password security, pattern recognition, and basic defensive cybersecurity concepts.
